# 数据库设计

## 用户表

主要用于用户登录，系统需求简单，仅提供 id、用户名和密码。

```SQL
CREATE TABLE user
(
    `id` INTEGER PRIMARY KEY,
    `username` VARCHAR(255) NOT NULL,
    `password` VARCHAR(255) NOT NULL
);
```

## 标签表

存储系统内的人工定义标签，用户可随意新增标签。

```SQL
CREATE TABLE tag
(
    `id` INTEGER PRIMARY KEY,
    `name` VARCHAR(255) NOT NULL
)
```
